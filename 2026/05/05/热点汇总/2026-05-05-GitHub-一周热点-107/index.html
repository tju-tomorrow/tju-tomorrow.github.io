GitHub 一周热点第107期

发表于 2026-05-05 分类于 热点汇总
本期看点：被OpenAI收购的AI安全工具promptfoo、AI代理事务所agency-agents、OpenClaw技能宝库、Claude Code插件claude-hud，以及字节开源的上下文数据库OpenViking。

视频链接：https://v.douyin.com/alHEIveTBx8/

1. promptfoo — AI 安全工具

用于评估和红队测试大模型应用的 CLI/库工具。

  • 功能：自动评估提示词和模型、比较不同模型差异、红队测试与漏洞扫描、CI/CD 自动化安全检测、发现代码安全合规问题
  • 支持 WebUI 和 CLI 两种使用方式
  • 2026年3月9日被 OpenAI 正式收购，整合进 OpenAI Frontier（企业级智能体协作平台）
  • 既是好用的安全工具，也是 AI 创业成功案例

2. agency-agents — AI 代理事务所

一个精心设计的 AI Agent 设计盒/人才市场，不是具体工具。

  • 每个 Agent 有独特专业领域、性格特点和工作流程（前端开发、后端架构、移动开发、AI 工程师等）
  • 每个角色有完整定义文档：身份定位、专业能力、工作流程、交付标准、沟通风格
  • 推荐配合 Claude Code 使用，直接复制 Agent 文件夹即可

3. awesome-openclaw-skills — OpenClaw 技能宝库

从社区 13000+ 个 Skill 中筛选的高质量技能合集。

  • 过滤掉 7000+ 低质量/恶意插件，保留 5400+ 个高质量 Skill
  • 按类别清晰整理：连接外部服务、开发工具、生产力工具、AI 相关等
  • 弥补了官方 Hub 只能用关键词搜索、体验一般的不足
  • 推荐搭配 SkillWise 做安装前安全审查

4. claude-hud — Claude Code 插件

显示 Claude Code 当前运行状态的插件。

  • 显示上下文使用情况、活跃工具、运行中的 Agent、待办事项进度
  • 使用 Claude Code 原生状态栏 API，无需单独窗口、无需 tmux
  • 安装：先装 Marketplace 插件 → 装 claude-hud → 配置 → 重启

5. OpenViking — 上下文数据库

字节跳动开源的 AI Agent 上下文数据库。

  • 解决核心问题：Agent 的记忆、资源、技能如何统一管理，放入统一的 Context 层
  • 配合 OpenClaw 使用，原生记忆效果提升 43%，输入 Token 成本降低 91%
  • 不是模型也不是 Agent，而是基础设施层的上下文数据管理方案

a point should be explicit .You as a user ,the browser like Chrome is on your side ,it fights for you.And A server is a provider that offers services . It can be visualize as a computer being open all the time 24/7.A

So domain port or ip these are the properties that servers apply on A website.

IP domain port

So 域名/domain shows up .域名就是 example.com

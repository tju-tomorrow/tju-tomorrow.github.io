每周编码时间

首先，设置每周编码时间统计：

注册 WakaTime 获取 WakaTime API Key

仓库中添加 Secret: WAKATIME_API_KEY

  • 注册 WakaTime

让我帮你设置 WakaTime：

  1. 安装 IDE 插件

    • 对于 VS Code：
      1. 打开扩展市场（Ctrl+Shift+X）
      2. 搜索 “WakaTime”
      3. 点击安装

  2. 获取 API Key：

    1. 注册/登录 WakaTime
    2. 进入 Settings → API Key
    3. 复制你的 API Key

  3. 配置 IDE 插件：

    • 第一次使用时，IDE 会提示输入 API Key
    • 粘贴你刚才复制的 Key
    • 等待几秒完成验证

  4. 开始使用：

    • 插件会自动追踪你的编码时间
    • 统计数据会显示在 WakaTime 仪表板
    • 包括：
      • 编码时长
      • 使用的语言
      • 项目时间分配
      • 编辑器使用情况

  5. 在 GitHub Profile 显示统计：

    在你的 profile README 中添加：

  1. 添加 GitHub Action：

  2. 添加 Secrets：

    • 在你的 profile 仓库中：
      1. Settings → Secrets and variables → Actions
      2. 添加 WAKATIME_API_KEY
      3. 添加 GH_TOKEN（GitHub Personal Access Token）

等待 24 小时后，你就能在 GitHub Profile 看到编码统计了！

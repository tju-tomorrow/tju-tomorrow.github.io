1.实验内容

1.实现无表头节点的链栈类MyStack

l 栈初始化, MyStack();//创建栈

l 销毁栈,~MyStack();//清除栈内元素,销毁栈

l 入栈, bool push(const int&);//入栈,成功返回true,失败返回false

l 出栈, bool pop(int& item); //出栈,成功返回true,失败返回false,出栈元素放在item

l 返回栈顶元素bool getTop(int& item);//获取栈顶,成功返回true,失败返回false,栈顶元素放在item

l 输出栈内元素,void printStack();//按照栈顶到栈底的顺序输出栈内所有元素

2.实现顺序循环队列类MyQueue

l 队列初始化, MyQueue(const int& capacity);//创建队列,容量为capacity

l 销毁队列,~ MyQueue();//销毁队列

l 入队,bool enQueue(const int&);//入队,成功返回true,失败返回false

l 出队,bool deQueue(int& item); //出栈,成功返回true,失败返回false, 出队元素放在item

l 返回队头元素,bool getFront(int& item); //获取队首,成功返回true,失败返回false,栈顶元素放在item

l 输出队列内元素, void printQueue();//按照队头到队尾的顺序输出栈内所有元素

3.基于MyStack进行扩展改进(如改变栈内元素类型、对符号进行合理数值映射)，实现中缀表达式转后缀表达式和表达式求值，支持小括号、加减乘除四则运算,一位正整数(0-9)运算,不涉及多位数、负数及小数

l bool postfixExpression(const string&, string& result); //中缀表达式转后缀表达式,转化成功返回true, 后缀表达式放在result中,表达式错误返回false, 错误字符串Expression is wrong!放在result中

l bool expressionVal(const string&, int& result);//表达式求值,表达式正确成功计算返回true,结果放在result中,表达式错误计算不能完成返回false, result中放0
